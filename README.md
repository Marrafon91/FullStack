# 🛒 DSCommerce

Sistema de **E-commerce Full Stack** desenvolvido com **Spring Boot**, **React** e **TypeScript**.

O projeto foi criado com o objetivo de colocar em prática conceitos modernos de desenvolvimento Full Stack, como arquitetura em camadas, autenticação com JWT, APIs REST, consumo de serviços HTTP, componentização e integração entre Front-end e Back-end.

# 🚀 Como executar

## Clonar o projeto

```bash
git clone https://github.com/Marrafon91/DSCommerce.git
```

## Backend

```bash
cd backend

./mvnw spring-boot:run
```

ou

```bash
mvn spring-boot:run
```

Servidor:

```
http://localhost:8080
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

Aplicação:

```
http://localhost:5173
```

# 🎯 Objetivo do projeto

O objetivo deste projeto é consolidar conhecimentos em desenvolvimento **Full Stack**, utilizando tecnologias amplamente adotadas pelo mercado.

Durante o desenvolvimento foram aplicados conceitos como:

* Arquitetura em camadas
* APIs REST
* Autenticação JWT
* Componentização
* Consumo de APIs
* Boas práticas de programação
* Separação de responsabilidades
* Tipagem com TypeScript
* Gerenciamento de estado
